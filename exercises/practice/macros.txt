# ============================================================================
# Cwiczenie: Makra (macros)
# ============================================================================
#
# Instrukcje:
#   - qq     - rozpocznij nagrywanie makra do rejestru 'q'
#   - q      - zakoncz nagrywanie
#   - @q     - odtwórz makro z rejestru 'q'
#   - @@     - powtorz ostatnie makro
#   - 10@q   - odtwórz makro 10 razy
#   - :reg q - pokaz zawartosc rejestru q (czyli makro)
#
# Kluczowa zasada: makro powinno konczyc sie na poczatku NASTEPNEJ linii (j0)
# aby mozna je bylo powtarzac!
#
# ============================================================================
