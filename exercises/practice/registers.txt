# ============================================================================
# Cwiczenie: Rejestry (registers)
# ============================================================================
#
# Instrukcje:
#   - :reg         - pokaz zawartosc rejestrow
#   - "ayy         - kopiuj linie do rejestru 'a'
#   - "ap          - wklej z rejestru 'a'
#   - ""           - domyslny rejestr (unnamed)
#   - "0           - ostatni yank (nie delete!)
#   - "1-"9        - historia usuniec (od najnowszego)
#   - "+           - systemowy clipboard
#   - "*           - selection clipboard (X11) / clipboard (macOS)
#   - "_           - black hole register (usun bez zapisu)
#   - "/           - ostatnie wyszukiwanie
#   - ".           - ostatni wpisany tekst
#   - ":           - ostatnia komenda ex
#   - "%           - nazwa biezacego pliku
#   - Ctrl-r {reg} - wklej rejestr w trybie Insert/Command
#
# ============================================================================
